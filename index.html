The Novocomum Table

Coffee table in burr oak characterized by three tops. The Novocomum Table was part of the furniture designed by the Italian designer for the headquarters of the Federation of Farmers, building Novocomum. The cost of this table is about 2.000 Euro.

This is the REAL table:

And this is the table shape that I've draw:

For this shape I've decided to produce it with a final function that draw all the model, inside this function there are some basilar component (cuboid and closed cylinders) created using the parameters gived to the function.

To realize this shape I've used this functions:

  1. Circle
  2. Cylinder (edited for obtain close cylinders)
  3. Cuboid
  4. Color
  5. T
  6. R
  7. S
  8. Novocomum (writed for parameterize the shape)

The Novocomum function has as parameters the radius and the height of the table, then it produce a shape of the table.

This is the code of the edited version of the cylinder function:

function CYLINDER(r,h){
var domain1 = DOMAIN([[0,2*PI], [0,r]])([36,1]);
var mapping = function(v){
var a = v[0];
var r = v[1];
return [r*COS(a), r*SIN(a)];
};
var base = MAP(mapping)(domain1);
var domain2 = DOMAIN([[-2*PI,2*PI]])([70]);
var crc = circle(r);
var model = MAP(crc)(domain2);
var cil = EXTRUDE([h])(model);
var cylinder = STRUCT([base,cil,T([2])([h])(base)]);
return cylinder;
};

The Chair for Sant'Elia nursery school

For this model I've decided to draw all the model proceeding step-by-step starting from the top and ending to the bottom of the chair. The majority of the chair's support is realized using Bezier's curves.

To realize this shape I've used this functions:

The most difficult part of this chair shape, was to draw all the support for the chair. I've realized the back support for the backrest using Rotational_surface:

//BACK SUPPORT

var p1 = [[2,0,2],[1.98,0,1.85],[1.78,0,1.85],[1.8,0,2]]
var c1 = BEZIER(S0)(p1)
var mapp1 = ROTATIONAL_SURFACE(c1)
var s1 = MAP(mapp1)(rot_domain)


var p2 = [[2,0,2],[1.98,0,2.15],[1.78,0,2.15],[1.8,0,2]]

The rest of the chair's support was realized using Bezier, because it's not possible to draw the support by turning it on a specific angle but it's necessary to draw it point-to-point with Bezier curves like this:

A Library prototype

This prototype was drawing by Terragni, but never physically realized. This was the last Terragni's project before his death.

This is the REAL library:

And this is the library shape that I've draw:

For this shape, I've decided to draw it using some functions created by me (Book_Spaces and Lateral_Shelfs) for drawing the majority of the model. The library legs are realized using Bezier's curves.

To realize this shape I've used this functions:

  1. Cuboid
  2. Color
  3. T
  4. R
  5. Bezier
  6. Intervals
  7. Rotational_surface
  8. Book_Spaces (created for draw the spaces for the books)
  9. Lateral_Shelfs (created for draw the lateral shelfs)

For this shape, I've created two new functions called:

  • Book_Spaces 

    //Function that draw the library's book spaces

function BOOK_SPACES(panel,n){
var shelf_h1 = T([0,1])([2,-1])(CUBOID([5.1,0.1,1.5]))
var level_1 = STRUCT([shelf_h1,panel])
for (var i = k ; i >= 0; i--) {
    level_1 = STRUCT([level_1,T([0])([i])(panel)])
};
return level_1
}

function LATERAL_SHELFS(shelf_h0,n){
